# 人生重开模拟器插件

一个基于Yunzai Bot的文字冒险游戏插件，让你体验不同的人生轨迹，感受命运的无常与多彩。

## 🎮 功能特色

- **自定义属性分配**：自由分配颜值、智力、体质、家境、快乐五项属性
- **随机人生模式**：一键随机生成属性，体验未知人生
- **丰富事件系统**：包含天赋、随机事件、人生转折点等
- **可视化结果**：生成精美的人生轨迹图片
- **多人生成**：支持多次重开，体验不同人生

## 📦 安装方法

### 前置要求

- Yunzai Bot
- Node.js 16+

### 安装步骤

1.将插件文件夹放入Yunzai的plugins目录，或者在Yunzai根目录执行下列命令克隆本仓库：

```bash
git clone --depth=1 https://github.com/Temmie0125/life-restart-plugin ./plugins/life-restart
```

2.重启Yunzai Bot

## 🎯 使用方法

### 基础命令

| 命令 | 说明 | 示例 |
|------|------|------|
| `#人生重开 5 5 5 5 0` | 自定义分配属性点 | `#人生重开 4 4 4 4 4` |
| `#随机人生重开` | 随机分配属性点 | `#随机人生重开` |
| `#人生重开帮助` | 查看详细使用说明 | `#人生重开帮助` |

### 属性说明

游戏提供五项基本属性，每项属性范围为0-10点：

| 属性 | 代号 | 说明 | 影响 |
|------|------|------|------|
| 颜值 | CHR | 外貌吸引力 | 社交、恋爱、机会 |
| 智力 | INT | 学习理解能力 | 学业、工作、解决问题 |
| 体质 | STR | 身体健康程度 | 寿命、健康、体力活动 |
| 家境 | MNY | 家庭经济条件 | 初始资源、教育、起点 |
| 快乐 | SPR | 幸福感程度 | 生活满意度、心理状态 |

### 属性点分配规则

- 总属性点：**20点**（固定）
- 单项属性范围：0-10点
- 分配必须用尽所有点数

### 使用示例

1.**平衡型人生**（推荐新手）：

```bash
#人生重开 4 4 4 4 4
```

2.**天才路线**（高智力）：

```bash
#人生重开 2 8 3 3 4
```

3.**豪门路线**（高家境）：

```bash
#人生重开 3 3 3 8 3
```

4.**颜值路线**（高颜值）：

```bash
#人生重开 8 3 3 3 3
```

## 🖼️ 结果展示

游戏结束后，会生成一张包含以下信息的人生轨迹图片：

1. **属性总评**：各项属性的最终评分和评价
2. **天赋列表**：获得的天赋及其效果
3. **人生轨迹**：按年龄排列的重要事件
4. **享年统计**：最终的寿命和总结

## 🛠️ 自定义配置

### 修改总属性点

如需修改总属性点，可在 `LifeEngine.js` 中调整：

```javascript
// 默认总点数
this.propertyPoints = 20;
```

### 自定义模板

结果图片的HTML模板位于：

```bash
plugins/life-restart/templates/result.html
```

可以修改此文件来自定义输出样式。
